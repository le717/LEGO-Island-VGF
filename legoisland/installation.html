Installation

Installation on modern Operating Systems

LEGO Island can be played on a number of modern PC operating systems. I've tried it on XP, Windows 7, Windows 8.1, and Windows 10 with very few issues. Some issues you may come across are bad click-detection where only one face of a LEGO element becomes clickable when the whole object should be, as well as crashes before the final credits screen.

Important notes about installation:
  • If you copy the CD contents to your hard drive, you can skip the No CD Crack below, as the installer will set the CD path to the folder you installed from. Just make sure you have the LEGO Island CD files in a permanent location!
  • Make sure to use the default installation path provided! I tried installing the game onto a secondary drive in a custom folder and the game would crash.
  • Find the installation directory, and right-click on each of these three files individually: CONFIG.EXE, ISLE.EXE, LEGOIsle.exe. Change the Compatibility mode to Windows XP (Service Pack 3)
  • Run CONFIG.EXE, open the Advanced Settings, and choose RGB Emulation or Direct3D HAL

Play the game wihtout the CD

Ok, so it's not exactly "cracking" being done... you just have to change a registry value to make the game think the CD drive is actually a directory in your computer. Follow the steps below at your own risk! WARNING: If you've never edited your registry before, be very careful! I will not be held responsible if you cause any problems!

  1. Once the game is installed from the disk, check to make sure that the game works on your system.
  2. Copy the contents of the CD to anywhere in your computer. For this example, I'm putting them in the same folder as LEGO Island was installed in a subfolder I created called "CD"
  3. Open up regedit.exe. If you don't know how to do this, open up the Start menu and find Run..., choose it, and type in regedit.
  4. From here,
    1. if you have a 32-bit operating system (or you don't know what you have), try navigating to:
      HKEY_LOCAL_MACHINE > SOFTWARE > Mindscape > LEGO Island
    2. if you have a 64-bit operating system (or you couldn't find the Mindscape folder in the last point), try navigating to:
      HKEY_LOCAL_MACHINE > SOFTWARE > Wow6432Node > Mindscape > LEGO Island
    3. if you've tried both of the steps above with no luck, try Edit > Find or CTRL+F in regedit and look for Mindscape. Once you find it, open up the LEGO Island folder.
  5. With LEGO Island selected, you should see a key called cdpath. Double-click on it.
  6. Now, all you need to do is link up this path to the path where you copied the CD's contents.
    1. If you have a 32-bit operating system and put the CD folder inside your LEGO Island folder, replace the cdpath's value from it's current drive letter to C:\PROGRA~1\LEGOIS~1\CD
    2. If you have a 64-bit operating system and put the CD folder inside your LEGO Island folder, replace the cdpath's value from it's current drive letter to C:\PROGRA~2\LEGOIS~1\CD
  7. Tada! No more CD needed!
